Ahmia blocks websites that contain child sexual abuse material. You can use this to filter child sexual abuse material from your own search indexes. Onion addresses are masked by creating an MD5 sum of the onion domain. For example, if the onion is abcdefghijklmnop.onion, the resulting hash would be 2ab597f404a139079640b2392d0cb593.
To calculate the MD5 checksum from an onion address, run the following command:
echo -n "abcdefghijklmnop.onion" | md5sum

Or using Python3:
import hashlib
hashlib.md5("abcdefghijklmnop.onion".encode('utf-8')).hexdigest()
