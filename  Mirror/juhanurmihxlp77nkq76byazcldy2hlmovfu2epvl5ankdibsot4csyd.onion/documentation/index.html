Documentation

The source code of Ahmia is open source and available on Github

You are free to build your own index and crawler.

The source code is mainly divided into three parts:

  • ahmia-index
    Elasticsearch index
  • ahmia-crawler
    Onion crawler
  • ahmia-site
    The search website
